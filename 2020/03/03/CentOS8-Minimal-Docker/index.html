3. Set Firewall#

  • RHEL 계열 리눅스들은 기본적으로 방화벽을 통해 포트가 막혀있음
  • 각 서비스를 외부에서 사용 하려면 방화벽 포트를 열어줘야 함
  • 자주 사용되는 서비스는 서비스 이름으로 방화벽 예외 항목에 추가하는 것이 가능
  • 다른 포트로 변경해서 쓰고 있다면 포트 번호로 추가해야 함
